# 🌿 NeoCampo - Sistema Integrado de Gestão Inteligente

## 📖 Sobre o Projeto
O **NeoCampo** é um ecossistema digital desenvolvido como projeto final integrador, consolidando soluções de IoT, Ciência de Dados, Visão Computacional e Computação em Nuvem para otimização da gestão agrícola. O sistema visa monitorar a saúde da lavoura, automatizar a irrigação e garantir a tomada de decisão baseada em dados.

## 🚀 Funcionalidades Principais
- **Planejamento Agrícola:** Gestão de área e insumos com banco de dados estruturado.
- **Monitoramento IoT:** Leitura em tempo real de sensores de umidade e automação de bombas.
- **Machine Learning:** Modelo preditivo para otimização do tempo de irrigação.
- **Visão Computacional:** Detecção de pragas via rede neural (YOLO).
- **Integração Cloud:** Disparo de alertas inteligentes via AWS SNS.


## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python
- **Interface:** Streamlit
- **Banco de Dados:** SQLite
- **Cloud:** AWS (SNS, IAM)
- **IA/ML:** Scikit-Learn, YOLO (Visão Computacional)
